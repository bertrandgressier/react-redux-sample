## React Redux Sample

Use :
- React-bootstrap
- Redux
= Redux-router
- redux-observable
- rxjs

This sample illustrates react with simple use case :
- Authentication
- Protected a page
- make async call and use redux to control the flow

Files are organized by feature and not by technical layer.
